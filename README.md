# parking-management-system-java

Java-based parking management system implementing pricing logic, input validation, discounts, and professional console receipt formatting.

##  Description
A console-based Java application that simulates a parking billing system.

##  Features
- Vehicle-based pricing (Car, SUV, Truck, Electric Vehicle)
- Input validation for user entries
- Surcharge for long parking duration
- Membership discounts and extra discounts
- EV charging fee support
- Formatted receipt output (professional console UI)
- End-of-day summary report

## Technologies
- Java
- Scanner (user input)
- Control structures (loops, conditionals, switch)
- Modular design (methods)
